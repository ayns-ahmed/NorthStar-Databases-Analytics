# NorthStar Urban Mobility Analytics

## Overview
This project presents a complete data analytics solution for a logistics-based urban mobility system. It integrates SQL, R, Python, and MongoDB to process, analyse, and optimise operational data.

## Repository Structure
- notebooks/01_sql_in_r.ipynb  
  → SQL queries executed using R (sqldf), including optimisation comparisons  

- notebooks/02_r_analytics.ipynb  
  → Data cleaning and visual analytics using ggplot2  

- notebooks/03_python_processing.ipynb  
  → Data preprocessing, feature engineering, and visualisation using pandas & seaborn  

- notebooks/04_mongodb_development.ipynb  
  → MongoDB Atlas integration, document design, CRUD operations, aggregation pipelines, and indexing  

## Technologies Used
- R (sqldf, ggplot2)  
- Python (pandas, numpy, matplotlib, seaborn, pymongo)  
- MongoDB Atlas (NoSQL database)  
- Google Colab  

## Key Features
- Multi-source data integration  
- Feature engineering for delivery analytics  
- Performance analysis using visualisations  
- NoSQL document modelling and querying  
- Query optimisation using indexing
